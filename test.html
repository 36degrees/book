Welcome to CSE 404!

This class will teach you how the web works by building a complete web browser from scratch. You will learn HTTP, HTML, CSS, and JavaScript, and build a complete working implementation of each of them.

Logistics

The class will run every Monday, Wednesday, and Friday 10:00-11:00.

  • Pavel Panchekha (pavpan@cs.washington.edu)

Topics

This course will cover every component of a basic web server:

  • The networking stack, including the HTTP protocol.
  • HTML parsing, including constructing the DOM.
  • CSS styling, including selecting elements and cascading rules.
  • Rendering, including padding, margins, fonts, and colors.
  • Text, including line breaking.
  • JavaScript, including events, DOM manipulation, and cookies.
  • The GUI, including browser history.

You will implement all of them from scratch and with minimal supporting libraries, with the exception of the JavaScript interpreter, for which we will be using V8.

Homework

Every homework will be done in Python 3 using Tk.

  • Network: Use Telnet to connect to a web server; dump page text to screen.
  • Text: create a GUI and write formatted text to it using a stateful lexer.
  • Layout: render paragraphs, break lines, and scrolling.
  • Styles: add support for attributes, colors, horizontal rules, and headings.
  • HTML: add support for margins by creating an HTML tree and rendering from it.
  • CSS: parse CSS and apply it.
  • GUI: draw a back and forward button, and add support for links.
  • HTTP: support web forms, including both GET and POST requests.
  • JavaScript: integrate V8, fire JS events, and handle DOM updates.
